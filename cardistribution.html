Car Distribution Website

The project idea I created for a vehicle distribution company. It include admin site roles : Ceo, Manager, Vehicle Specialist, Content Writer, and user roles : Guest and Member. Let's see more below:

Client Side

Home Page

Its home page has background video, a warmly welcome caption and attracting user to become a member.

About Page

Its home page has background video, a warmly welcome caption and attracting user to become a member.

Contact Page

Its home page has a combination of an image and grandient backgrund and a very simple contact detail.

Notification Session

It include a Notification system. When server side upload a new car, a notification will send to memebers. Members can know by finding out noti bell has badge.

Vehicle List Page

All vehicles are divided by paginations, 12vehicles by each pagination. User can search by brands, category, price or by typing in search box.

Vehicle Services

User can choose regularservices and customized services.

Server Side

User Listings

Admin can see member lists and give them permissions or remove them. Member can be searched by roles ,name or email.

Brand & Category Page

Admin can add and remove brand and category.

Vehicle Page

Admin can import new cars, edit its details and remove from list. It also has a filter. Admin can search cars by brands, category, price and model. So convenient for admin I guess.

Positions Page

Admin can create new positions althought it need developer to make some changes. He can add more job positions.

Services Page

Admin can decide services, adding, removing and editing as he want.
