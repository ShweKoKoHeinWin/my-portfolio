E-commerce Shopping Website

This is a project from Step By Step youtube channel. I just modified it to what I wanted. I am very thankful to her by making such a wonderful tutorial. I had learned alot from this project. Hope you will like it. Detail is just below.

Client Side

Home Page

This page display shop items randomly every time user reload. Users can search items directly by search bar or brands and categories.

About Page

It has a cart system. Users can add items to cart, update the quantity and make orders.

Profile Page

In profile page, user can edit his or her identy, see orders and comfirm orders.

Server Side

Productlists

Admin can create, update, remove items and he can see all products.

Orders lists

Admin can see all orders and invoices.
